## ceroot-modelinfo
think模型解析模块  
> 以下类库都在`\\ceroot\\modelinfo`命名空间下

> Quiet   静态解析类

> Syserm  系统模型解析类

安装
> composer require ceroot/think-modelinfo

自定义模型信息参考

使用
> 实例化
```php
modelinfo();
```

> 列表
```php
modelinfo()->getList($data);
```

> 添加
```php
modelinfo()->getAdd($data);
```

> 编辑
```php
modelinfo()->getEdit($data);
```
